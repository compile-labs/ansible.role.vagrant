# Ansible Role Vagrant

An Ansible role for installing Vagrant. over Linux and Mac OS X.

## Role Variables

- `vagrant_version` - Vagrant version (default: `1.6.5`)
- `vagrant_arch` - Vagrant architecture build (default: `x86_64`)
